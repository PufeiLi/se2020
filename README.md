### 界面
* (1)界面有一个棕色输入框和一个灰色输出框，分别用于三角函数变量的输入和值的输出。
* (2)界面有四个按钮，分别为sin，cos，tan，cot，点击四个按钮可以分别计算对应三角函数的值。
